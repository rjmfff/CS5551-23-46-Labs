Home page is the very first page the user is presented upon successful login into an application. The content presented on this page may vary across applications. The basic idea is to enable the user to start using the application from this page. This page may provide the user with general welcome notation and application description. It shall also include links to several different aspects of the applciation, either directly or using a Menu bar. It also provides the user with a logout option to exit out of the application.
